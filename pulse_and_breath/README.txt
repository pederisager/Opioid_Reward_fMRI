Folder contains pulse and respiration data, collected with Phillips wireless PPU unit during all fMRI scans.


##########
Key facts
##########

- Sampling rate: 500hz
- Empty line exept a hash (#) marks begnning of scan
- ReadPhilipsScanPhysLog does not provide the number of peaks, but contain the data needed to do so. The findpeaks function in matlab should take care of the peak detection.
